🎥 DESCRIPTION

Proper 3rd Person Camera for Crimson Desert is a carefully tuned camera adjustment designed to make the game feel more cinematic, grounded, and immersive.

Inspired by the third-person camera style of Red Dead Redemption 2, this mod focuses on improving the feel of the default gameplay camera without turning it into a full overhaul or preset pack.

✨ What it improves:
- Lowers and stabilizes the default camera position
- Makes movement feel smoother and less “floaty”
- Enhances horse riding camera behavior
- Slightly widens the near third-person / pseudo first-person view
- Keeps the original camera distance switching system intact

The goal is simple:
👉 a more natural, cinematic, and controlled third-person experience while staying faithful to the original game feel.
